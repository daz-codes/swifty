## Number: {{ number }}

Swifty is the next generation of static site generator.

It makes it really simple to create a full website that loads pages almost instantly.

## Quick Start

Install Swifty:

```
npm install swifty
```

Start a new project:

```
npm swifty:create
```

Add a template

Add an index.md to the pages folder

Add another page

Start the server

npm start
